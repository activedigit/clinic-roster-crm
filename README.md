# 🩺 מערכת לניהול משמרות רופאים ואחים/ות

תוכנה חודשית לניהול משמרות (תורנויות) של הצוות הרפואי:

- **כל רופא/אח/ות** מקבל **קישור אישי** שנפתח מהטלפון, ובו הוא מסמן את הימים שבהם הוא רוצה לעבוד (נשמר אוטומטית).
- **המנהל** מקבל **לוח ניהול** עם טבלה חודשית של כל העובדים, יכול לערוך ולמחוק כל משמרת בלחיצה, להוסיף עובדים, לשלוח קישורים בוואטסאפ, ולהדפיס או לייצא את הטבלה.

התוכנה בנויה ב-**Node.js בלבד, ללא ספריות חיצוניות** — צריך רק דבר אחד: Node מותקן על המחשב.

---

## ▶️ הפעלה

1. התקן [Node.js](https://nodejs.org) (גרסה 16 או חדשה יותר) אם אינו מותקן.
2. פתח את שורת הפקודה (PowerShell) בתוך תיקיית התוכנה והרץ:

   ```powershell
   node server.js
   ```

3. יודפס לך על המסך **קישור הניהול הסודי**, לדוגמה:

   ```
   http://localhost:3000/admin/XXXXXXXXXXXX
   ```

   פתח אותו בדפדפן ושמור אותו (Bookmark). **הקישור הזה סודי — אל תשתף אותו.**

> 💡 לעצירת התוכנה: הקש `Ctrl + C` בחלון הפקודה.

---

## 👥 אופן השימוש

1. מלוח הניהול ← לשונית **«עובדים»** ← הוסף כל רופא/אח/ות (שם + תפקיד + מספר וואטסאפ אופציונלי).
2. לכל עובד יופיע **קישור אישי**. לחץ **🟢 וואטסאפ** כדי לשלוח אותו ישירות, או **📋 העתק קישור**.
3. העובד פותח את הקישור מהטלפון, בוחר ימים, והם מגיעים אליך אוטומטית.
4. מלשונית **«לוח חודשי»** תראה את כולם, תוכל ללחוץ על כל תא כדי להוסיף/להסיר משמרת, לעבור בין חודשים, ולהדפיס או לייצא לאקסל.

---

## 📱 גישת העובדים מהטלפונים

הקישורים פועלים בהתאם לאופן שבו אתה פותח את לוח הניהול:

- **אותה רשת Wi-Fi (מרפאה/בית חולים):** פתח את לוח הניהול דרך הקישור `http://<כתובת-IP>:3000/admin/...` (מופיע בהפעלה תחת «לכניסה ממכשיר אחר»). אז קישורי העובדים ישתמשו באותה כתובת וייפתחו בטלפונים המחוברים לאותה רשת.

- **מכל מקום באינטרנט:** יש לפרסם את התוכנה על שרת (כמו Render, Railway או VPS), או להשתמש במנהרה זמנית כגון:

  ```powershell
  npx cloudflared tunnel --url http://localhost:3000
  ```

  תקבל קישור ציבורי זמני; פתח דרכו את לוח הניהול, ואז קישורי העובדים יהפכו לציבוריים וייפתחו מכל מקום.

---

## 💾 נתונים וגיבוי

כל הנתונים נשמרים בקובץ אחד:

```
data/db.json
```

- לגיבוי: העתק את הקובץ הזה.
- בתוכו נמצא גם **קישור הניהול הסודי** (`adminToken`). אם תרצה לשנות אותו, מחק את הקובץ וייווצר קישור חדש בהפעלה (שים לב: מחיקת הקובץ מוחקת את כל העובדים והמשמרות).

---

## ⚙️ שינוי הפורט (Port)

ברירת המחדל היא פורט `3000`. לשינוי:

```powershell
$env:PORT = 8080; node server.js
```
